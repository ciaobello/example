*My GitHub Page # Installation Die folgenden Abschnitte erklären die Installation und Aktualisierung von Contao. Sie lernen wie man das Contao-Installtool verwendet, wie man eine Installation per FTP synchronisiert, wie man den Safe Mode Hack einrichtet und wie man einen Webserver für Contao konfiguriert. ## Contao installieren Laden Sie sich zunächst das [aktuelle Contao-Archiv][1] herunter und entpacken Sie es auf Ihrem lokalen Rechner. Verschieben Sie die Dateien in das `htdocs`-Verzeichnis Ihrer XAMPP-Installation oder kopieren Sie sie mit einem FTP-Programm (z.B. [WinSCP][2]) auf Ihren Server. Der Basisordner heißt wahrscheinlich `htdocs`, `httpdocs`, `html` oder `public_html`. Wenn Sie SSH-Zugriff auf den Server haben, können Sie alternativ folgende Befehle verwenden: ``` {.bash} curl -L http://download.contao.org | tar -xzp ```
